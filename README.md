# Disease Trend Analysis Using Pandas and SQL

## Overview
This project analyzes healthcare data to find disease trends over time. The dataset was cleaned and prepared in Pandas, and SQL was used to calculate KPIs and extract insights.

## Objective
The main goal of this project is to understand how diseases change over time and identify patterns by age group, gender, and city.

## Tools Used
- Python (Pandas)
- SQL

## Work Done
- Cleaned missing and inconsistent values
- Standardized columns and data types
- Created new columns for analysis
- Performed KPI analysis in SQL
- Studied disease patterns over time

## Key Analysis
- Disease trend over time
- Disease count by gender
- Disease count by age group
- City-wise disease distribution
- Overall patient trends

## Files
- `data_cleaning.ipynb` – data cleaning and preparation in Pandas
- `sql_queries.sql` – SQL queries for KPI and insight analysis
- `dataset.csv` – dataset used in the project
